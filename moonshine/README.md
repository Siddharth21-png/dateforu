A romantic, interactive web experience that takes your special someone through a personalized date planning journey with continuous background music flowing seamlessly across all pages.

## Overview

Moonshine is a multi-page web application designed to create a memorable date planning experience. Users navigate through different stages - selecting a date, choosing food preferences, picking desserts, planning activities, and ending with a beautiful floral animation - all while enjoying uninterrupted background music.

## Features

- **Continuous Background Music**: Seamless MP3 playback across all pages without restarts or interruptions
- **Interactive Selection Pages**: 
  - Date picker
  - Food selection (burgers, biryani, pasta, pizza, dosa, and more!)
  - Dessert choices (boba, matcha, ice cream, etc.)
  - Activity planning (movie dates, Taj Mahal, parks, and local spots)
- **Beautiful Final Page**: Animated flower display with heartfelt message
- **Responsive Design**: Works perfectly on desktop and mobile devices
- **LocalStorage Integration**: Saves user selections and music playback position

## Technology Stack

Built with modern web technologies:
- **HTML5** - Semantic markup and structure
- **CSS3** - Custom styling for each themed page
- **Vanilla JavaScript** - Interactive functionality and music control
- **iframe Technique** - Continuous audio playback across page navigation
- **LocalStorage API** - Persistent data storage
